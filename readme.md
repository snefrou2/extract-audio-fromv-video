# 🎵 Audio Tools

Application desktop Electron permettant :

- Convertir des vidéos en WAV
- Convertir des fichiers audio :
  - MP3
  - WAV
  - M4A
  - FLAC
  - OGG
  - etc.

Fonctionne entièrement en local avec FFmpeg embarqué.

---

# ✨ Features

## Video → Audio

Convertit automatiquement :

```txt
mp4
mov
avi
mkv
webm
```

vers :

```txt
wav
```

---

## Audio Converter

Supporte :

```txt
mp3
wav
m4a
aac
flac
ogg
opus
wma
aiff
alac
ac3
amr
webm
mka
```

Conversion vers :
- WAV
- MP3
